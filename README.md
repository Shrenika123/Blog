Blogs

Tools used: Redux Antd

Functinalities: 
1)Blogs loaded and stored on central store 
2)Selcted Blog stored on central store 
3)on clicking on comments field comments api is called and stored on store 
4)When blog is not found an alert is trown 
5)When loading api's loader is shown 
6)When the page is not present page not found is shown
7)Debounce on search 
8)blog Routing

Note:As design was not provided have implemented in with simple design,while concentration on functinalities API used are mock ones
